# Telegram-Image-Processor-n8n

Hệ thống tự động hóa luồng công việc giữa Telegram Bot và Google Drive.

## Tính năng chính:
- Tự động tiếp nhận hình ảnh từ Telegram.
- Chèn nội dung (Caption) làm watermark lên hình ảnh.
- Lưu trữ file vào Google Drive với tên file tùy chỉnh.
- Gửi lại liên kết truy cập file trực tiếp cho người dùng.

## Cách cài đặt:
1. Tải file `workflow.json` từ repository này.
2. Mở n8n của bạn lên.
3. Import file `workflow.json` vào n8n.
4. Cấu hình Credentials cho Telegram và Google Drive.
5. Bật Workflow và tận hưởng!
